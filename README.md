# Multimodal-Liver-Disease-Detection
A multimodal deep learning framework for disease detection using CT images and clinical data.

## Key Features
- MobileNetV3-Small based classification
- Image preprocessing and enhancement
- Grad-CAM explainability
- PCA and t-SNE visualization
- HMM-based feature extraction
- Cross-validation evaluation

## Results
- Cross-Validation Accuracy: 97.99%
- Best Validation Accuracy: 95.81%
- Micro AUC: 0.9993
- Macro AUC: 0.9779

## Technologies
PyTorch, OpenCV, Scikit-learn, Albumentations, NumPy, Pandas
